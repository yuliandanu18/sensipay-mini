
Tambahkan route berikut ke routes/sensipay.php
(sesuaikan dengan struktur prefix & middleware yang sudah ada):

    use App\Http\Controllers\Sensipay\ParentManagementController;

    Route::middleware(['web', 'auth', 'role:owner,operational_director,academic_director'])
        ->prefix('sensipay')
        ->as('sensipay.')
        ->group(function () {
            Route::get('parents', [ParentManagementController::class, 'index'])
                ->name('parents.index');

            Route::get('parents/{student}/edit', [ParentManagementController::class, 'edit'])
                ->name('parents.edit');

            Route::post('parents/{student}', [ParentManagementController::class, 'update'])
                ->name('parents.update');
        });

Pastikan di model Student (app/Models/Student.php) sudah ada relasi:

    public function parentUser()
    {
        return $this->belongsTo(\App\Models\User::class, 'parent_user_id');
    }

Lalu:
- Admin / owner bisa membuka: /sensipay/parents
- Pilih siswa -> "Atur Orang Tua"
- Isi nama + email
- Sistem otomatis membuat / mengupdate user role parent dan menghubungkannya ke siswa.
